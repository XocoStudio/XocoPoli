# XocoPoli

**XocoPoli** es un potente modelador 3D poligonal diseñado para ejecutarse directamente en el navegador. Enfocado en el modelado "Low Poly" y "Hard Surface", ofrece un conjunto completo de herramientas para crear desde formas básicas hasta modelos complejos mediante manipulación de vértices, aristas y caras.

## Características Principales

### 1. Modelado Poligonal Completo
Trabaja a distintos niveles de detalle para tener control total sobre tu malla:
*   **Modo Vértice:** Selección y edición punto a punto.
*   **Modo Arista (Edge):** Manipula las líneas de conexión. Funciones avanzadas como "Bridge" (puente) y "Bevel" (biselado).
*   **Modo Cara (Facet):** Extruye, inserta (Inset) y separa caras para crear volumen.
*   **Modo Objeto:** Mueve, escala y rota objetos completos.

### 2. Herramientas de Modelado Avanzadas
*   **Extrude:** Crea nueva geometría empujando caras hacia fuera o hacia dentro.
*   **Inset:** Crea caras más pequeñas dentro de las seleccionadas.
*   **Loop Cut:** Añade anillos de aristas para subdividir la geometría.
*   **Knife:** Corta caras libremente para crear nueva topología.
*   **Bevel:** Suaviza bordes afilados creando chaflanes.
*   **Bridge:** Une dos caras seleccionadas creando un túnel o puente entre ellas.
*   **Spherify:** Transforma cubos en esferas perfectas.
*   **Subdivide:** Aumenta la resolución de la malla suavizando (Catmull-Clark) o linealmente.

### 3. Sistema de Materiales
*   **Standard:** Color sólido básico.
*   **Normal:** Visualiza las normales de la superficie (útil para depuración).
*   **Clay:** Acabado tipo arcilla para esculpido.
*   **Gold & Matrix:** Efectos visuales estilizados.
*   **Texturas:** Sube imágenes para usarlas como texturas sobre tus modelos.

### 4. Gestión de Escena
*   **Capas (Layers):** Organiza tu escena ocultando o mostrando partes en 8 capas diferentes.
*   **Primitivas:** Inicia tus modelos con Cubos, Planos, Esferas, Cilindros, Conos, Toros y Anillos.
*   **Gizmos:** Herramientas visuales intuitivas para trasladar, rotar y escalar.

### 5. Importación y Exportación
*   **OBJ & GLB:** Importa y exporta en formatos estándar de la industria.
*   **Historial:** Sistema robusto de Deshacer/Rehacer (Undo/Redo).

## Controles
*   **Click Izquierdo:** Seleccionar / Usar herramienta.
*   **Click Derecho + Arrastrar:** Orbitar cámara.
*   **Click Rueda (o Shift+Click) + Arrastrar:** Panorámica (Mover vista).
*   **Rueda del Ratón:** Zoom.
*   **Shift + Click:** Selección múltiple / Selección de caja (Box Select).
*   **Atajos de Teclado:**
    *   `1-4`: Cambiar modo (Objeto, Cara, Arista, Vértice).
    *   `Ctrl+Z / Ctrl+Y`: Deshacer / Rehacer.
    *   `G / R / S`: Activar gizmos de Traslación, Rotación y Escalado.
    *   `Supr`: Borrar selección.
